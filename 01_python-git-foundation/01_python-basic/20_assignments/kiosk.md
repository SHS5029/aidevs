# Assignment 08 - Amusement Park Kiosk

Python의 입력, 자료형, 조건문, 반복문, 리스트를 함께 사용해 놀이공원 탑승 안내 키오스크를 만듭니다.

## 과제 파일

```text
submissions/assignment08_amusement_park_kiosk.py
```

## 주제

방문자의 이름, 나이, 키를 입력받고 탑승할 놀이기구를 안내하는 콘솔 프로그램을 만듭니다.

사용자는 리스트에 저장된 놀이기구 메뉴를 확인하고 놀이기구를 여러 번 선택할 수 있습니다. 선택할 때마다 방문자의 나이와 키를 기준으로 탑승 가능 여부를 안내합니다.

## 학습 목표

```text
1. input()으로 문자열, 정수, 실수 값을 입력받습니다.
2. if / elif / else로 값의 범위를 판단합니다.
3. match-case로 정해진 메뉴를 처리합니다.
4. while True, break, continue로 반복 흐름을 제어합니다.
5. list에 저장된 값을 for 반복문으로 하나씩 활용합니다.
```

## 놀이기구

아래처럼 메뉴 번호가 포함된 놀이기구 문자열을 하나의 리스트에 저장합니다.

```text
1. 롤러코스터
2. 관람차
3. 어린이 기차
```

메뉴를 출력할 때 일반 `for` 반복문으로 리스트의 문자열을 하나씩 보여 줍니다.

## 탑승 조건

### 롤러코스터

```text
나이가 12세 이상이고 키가 140.0cm 이상이면 탑승할 수 있습니다.
나이가 12세 미만이면 나이 제한 안내를 출력합니다.
나이는 충족하지만 키가 140.0cm 미만이면 키 제한 안내를 출력합니다.
```

### 관람차

```text
나이가 6세 이상이고 키가 100.0cm 이상이면 탑승할 수 있습니다.
나이가 6세 미만이면 나이 제한 안내를 출력합니다.
나이는 충족하지만 키가 100.0cm 미만이면 키 제한 안내를 출력합니다.
```

### 어린이 기차

```text
나이가 12세 이하이고 키가 90.0cm 이상이면 탑승할 수 있습니다.
나이가 12세를 초과하면 연령 제한 안내를 출력합니다.
나이는 충족하지만 키가 90.0cm 미만이면 키 제한 안내를 출력합니다.
```

## 요구사항

### 1. 방문자 정보 입력

```text
1. 이름을 입력받아 문자열로 저장합니다.
2. 나이를 입력받아 정수로 변환합니다.
3. 키를 입력받아 실수로 변환합니다.
```

### 2. 나이대 분류

`if / elif / else`를 사용해 방문자의 나이대를 구분하고 출력합니다.

```text
0세 이상 13세 미만: 어린이
13세 이상 19세 미만: 청소년
19세 이상: 성인
```

### 3. 반복 메뉴

```text
1. while True로 놀이기구 선택을 반복합니다.
2. 매 반복마다 놀이기구 메뉴를 출력합니다.
3. 메뉴 번호 또는 q를 input()으로 입력받습니다.
4. q를 입력하면 break로 반복문을 종료합니다.
```

### 4. 메뉴 선택

```text
1. match-case로 1번, 2번, 3번 메뉴를 처리합니다.
2. 각 메뉴에서 해당 놀이기구의 탑승 가능 여부를 판단합니다.
3. 잘못된 값을 입력하면 case _에서 안내 문장을 출력합니다.
4. 잘못된 입력 뒤에는 continue로 다음 메뉴 선택으로 이동합니다.
```

### 5. 리스트 데이터 활용

```text
1. 메뉴 번호가 포함된 세 개의 놀이기구 문자열을 하나의 리스트에 미리 저장합니다.
2. 메뉴를 출력할 때 for 반복문으로 리스트의 값을 하나씩 꺼냅니다.
3. 리스트에서 꺼낸 문자열을 그대로 출력합니다.
4. 리스트에 새로운 값을 추가하거나 기존 값을 삭제하지 않습니다.
```

## 최종 문제 (대문제)

다음 소문제를 난이도 순서대로 해결하여, 서비스 흐름이 처음부터 끝까지 이어지는 놀이공원 탑승 안내 키오스크를 완성하세요.

### 기초 난이도

#### 소문제 1. 방문자 정보 입력하기

```text
1. 프로그램을 시작하면 방문자의 이름, 나이, 키를 차례대로 입력받습니다.
2. 이름은 문자열, 나이는 정수, 키는 실수로 저장합니다.
```

#### 소문제 2. 방문자의 나이대 안내하기

```text
1. 입력받은 나이를 기준으로 나이대를 판단합니다.
2. if / elif / else를 사용해 어린이, 청소년, 성인 중 하나를 출력합니다.
```

### 중급 난이도

#### 소문제 3. 놀이기구 메뉴 만들기

```text
1. "1. 롤러코스터", "2. 관람차", "3. 어린이 기차"를 하나의 리스트에 저장합니다.
2. for 반복문으로 리스트의 메뉴를 하나씩 출력합니다.
3. 리스트에 새로운 값을 추가하거나 기존 값을 삭제하지 않습니다.
```

#### 소문제 4. 반복 메뉴 구현하기

```text
1. while True를 사용해 놀이기구 선택 서비스를 반복합니다.
2. 반복이 시작될 때마다 놀이기구 메뉴를 출력합니다.
3. 사용자에게 1, 2, 3 또는 q 중 하나를 입력받습니다.
4. q를 입력하면 break로 반복문을 종료합니다.
```

### 심화 난이도

#### 소문제 5. 탑승 가능 여부 판단하기

```text
1. 1, 2, 3을 입력하면 match-case로 선택한 놀이기구를 구분합니다.
2. 각 놀이기구의 case 안에서 if / elif / else로 나이와 키 조건을 검사합니다.
3. 탑승할 수 있으면 탑승 가능 안내를 출력합니다.
4. 탑승할 수 없으면 나이 제한 또는 키 제한 안내를 출력합니다.
```

#### 소문제 6. 예외 메뉴와 전체 흐름 완성하기

```text
1. 잘못된 메뉴는 case _에서 안내한 뒤 continue로 메뉴 선택을 다시 시작합니다.
2. 정상적인 안내가 끝난 뒤에도 다음 놀이기구를 선택할 수 있도록 메뉴로 돌아갑니다.

```
